# Contributing to Agentic AI HTML5 Parser

Thank you for your interest in contributing to this project! We welcome contributions that help improve the parser and its agentic pipeline.

## How to Contribute

1.  **Fork the Repository**: Create your own fork of the project on GitHub.
2.  **Clone the Fork**: Clone your fork to your local machine.
3.  **Create a Branch**: Create a new branch for your feature or bug fix.
4.  **Make Changes**: Implement your changes and ensure they follow the project's coding standards.
5.  **Run Tests**: Ensure all existing tests pass and add new tests for your changes.
    ```bash
    python -m pytest tests/test_suite.py
    ```
6.  **Commit Changes**: Commit your changes with clear and descriptive messages.
7.  **Push to GitHub**: Push your branch to your fork on GitHub.
8.  **Submit a Pull Request**: Open a pull request against the main repository.

## Coding Standards

- Follow PEP 8 for Python code.
- Use meaningful variable and function names.
- Include docstrings for all classes and functions.
- Ensure all new features are covered by agents in the pipeline.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub with:
- A clear description of the problem.
- Steps to reproduce the issue.
- Expected vs. actual behavior.

Happy coding!
